# Creases :shirt:

Tools for efficiently folding markdown sections in Obsidian.

![creases-overview](https://user-images.githubusercontent.com/693981/156103767-33f311de-39ac-422d-b8ea-987ea9c63f7b.png)

Add `%% fold %%` markers to "crease" your markdown. Then run **Fold along creases** to get your file into the state you want.

- **Fold along creases** â€“ Fold all the creased sections of your file

- **Iron out the creases** â€“ Remove all the creases from the current file
- **Crease the current folds** â€“ Applies a crease to the current folds in the file, allowing you to easily get back to these current folds using **fold along creases**.
